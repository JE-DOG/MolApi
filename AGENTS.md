# Agent Context Index

Этот файл должен быть первой точкой входа для ИИ-агента в проекте `MolApi`.
Не загружай весь контекст сразу: прочитай только те файлы из `.agents/context/`,
которые относятся к текущей задаче.

## Как выбирать контекст

- `.agents/context/role-and-quality.md` - роль агента, стиль общения, стандарт качества кода,
  отношение к неопределенности и компромиссам. Читать в начале любой нетривиальной задачи.
- `.agents/context/project-structure.md` - структура репозитория, Gradle-модули, физические пути,
  пакеты и важные особенности именования. Читать перед изменением модулей, source sets или Gradle.
- `.agents/context/kmp-implementation-rules.md` - правила Kotlin Multiplatform-разработки:
  `commonMain`, платформенные ограничения, зависимости, source sets. Читать перед кодом в
  KMP-модулях.
- `.agents/context/molapi-contract.md` - публичный контракт библиотеки `dag.khinkal.molapi`,
  существующие core/http сущности и запреты на преждевременные фичи. Читать перед изменением
  `molapi/*`.
- `.agents/context/workflow-and-agents.md` - рабочий процесс, саб-агенты, TDD, ревью и поведение
  при блокерах. Читать перед крупными изменениями, багфиксами и задачами с параллельной работой.
- `.agents/context/build-and-verification.md` - команды сборки, тестов, проверки и правила работы
  с инструментами. Читать перед запуском Gradle, тестов или финальным отчетом.
- `.agents/context/integrations-and-secrets.md` - правила работы с MCP/интеграциями и секретами.
  Читать перед задачами с Kaiten, GitLab, Mobile MCP или любыми токенами.

## Базовые правила

- Сначала пойми задачу и найди минимальный релевантный контекст.
- Не меняй архитектуру без явного запроса или доказанной необходимости.
- Не придумывай факты. Если данных не хватает и это нельзя проверить локально, скажи прямо.
- Не записывай секреты, токены и приватные credentials в репозиторий.
- Не трогай `.gradle` напрямую. Для исходников зависимостей используй `ksrc`, начиная с
  `ksrc --help`.
- Для shell-команд в этом проекте предпочитай `rtk`-префикс.
